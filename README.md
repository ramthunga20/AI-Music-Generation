# AI Music Generation

## Description
This project generates MIDI music using Deep Learning and TensorFlow.

## Features
- MIDI dataset preprocessing
- Neural network training
- AI-based music generation
- MIDI output generation

## Technologies Used
- Python
- TensorFlow
- NumPy
- Music21

## How to Run

python preprocess.py
python train.py
python generate.py

## Output
